# 🎵 Spotify Music Data Analysis using SQL

## 📌 Project Overview

This project is an end-to-end SQL data analysis project built using the Spotify Music Dataset. The objective is to analyze song performance, artist popularity, album statistics, streaming trends, and audience engagement using SQL. The project demonstrates SQL concepts from beginner to advanced while solving real-world business problems through data-driven analysis.

---

## 🎯 Project Objectives

- Design and create a SQL database
- Import and validate Spotify music data
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Analyze artist, album, and song performance
- Solve business problems using SQL
- Generate actionable business insights

---

## 📂 Dataset Information

The dataset contains music-related information collected from Spotify and YouTube.

### Dataset Columns

| Column | Description |
|---------|-------------|
| Artist | Artist name |
| Track | Song title |
| Album | Album name |
| Album_Type | Album or Single |
| Danceability | Danceability score |
| Energy | Energy score |
| Loudness | Loudness level |
| Speechiness | Speech score |
| Acousticness | Acoustic score |
| Instrumentalness | Instrumental score |
| Liveness | Liveness score |
| Valence | Positivity score |
| Tempo | Song tempo (BPM) |
| Duration_min | Song duration (minutes) |
| Title | Video title |
| Channel | YouTube channel |
| Views | Total YouTube views |
| Likes | Total likes |
| Comments | Total comments |
| Licensed | Licensed content |
| Official_Video | Official video status |
| Stream | Spotify streams |
| EnergyLiveness | Calculated metric |
| Most_Played_On | Spotify or YouTube |
